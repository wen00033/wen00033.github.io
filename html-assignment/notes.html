The five html tag:

The form tag
The form tag is the paired tag. There are two important attributes called action and method. Action refers to the form's data where to send the message, and method defines how is the information sent. There are two values define the message: get for not sensitive information; post for privacy information.
The important text tag
"The important text we use" < strong> < /strong> to emphasize important information in the text.It also a paired tag.
The Horizontal Line tag
The horizontal line is a single tag, we use < hr> to represent a break in a content
The blockquote tag
This tag is a paired tag.Most time used for a quote like a great sentence in novel and a motto from an entrepreneur. We use < blockquote>and< /blockquote> on html.
Abbreviation tag
Abbreviation tag is used for words like "TV","ASAP".It can show the expanded version of definition ,like ASAP
